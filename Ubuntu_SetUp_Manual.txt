SGWグループウエア　Ubuntuへのインストールマニュアル SgwSetup2022_10

※Ubuntu16.04でセットアップ
※LAMPをインストールして、mysqlはその環境を利用したほうがよい
※データベースskt作成とskt.sqlからのデータ初期化は手動で行う
 #mysql -u root -p -e "create database skt;"
 #mysql -u root -p skt < skt.sql  
※SGW.warを使ってwebappsへコピーする

(1)ネットワークの設定
入力した内容が保存できないことがあります。そのときは、少し時間をおいてから再度
入力するとうまくいきます。
以下は,サーバのIpアドレスを172.16.129.204とした場合の例です。ネットワークの環境に合わせて入力します。
ipアドレス   172.16.129.204　　　
ネットマスク 255.255.255.0
ゲートウエイ 172.16.129.1
DNS  172.16.0.30
を設定


(2)セットアップ

CDから、SgwSetupフォルダを/home/userにコピーします。

コマンドライン(端末）で
sudo su

cd SgwSetup

chmod +x setup.sh

./setup.sh

 最初に、プロキシーを使用するかどうか聞いてきます。
プロキシーで認証が必要なときは、２を選択して ipaddress portを入力します。
以下は例です。ネットワークの環境に合わせて入力します。
172.16.0.31:12080
userとパスワードは
sakatoku3:sakatoku3
とします

press enter keyの表示が数回出ます。その際は、Enterキーを押してください。
y/nで聞かれた場合は、yes（y)を入力してください。
データベース（mysql)のパスワードも数回聞かれます。すべて同じパスワードを入力してください。


(5)テスト表示

http://172.16.129.204:8080/SGW/Loginでログインできます。IDをadmin、パスワードをpassとしてください。

・共有も動作するようにしています。
WindowsPCから「\\172.16.129.204」のようにIpアドレスを指定してアクセスします。
予定は、共有から、サーバのtenpuフォルダにあるyotei.xlsxというファイルを編集することで変更できます。

・sshも動作させていますので、teraterm等でメンテナンスが可能です。（ログインID,パスワードはUbuntuに設定したものと同じ）
